# 🤝 Guia de Contribuição

Ficamos muito felizes que você queira contribuir com o **Gestor de Despesas Consciente**! Para manter a organização e a qualidade do código, pedimos que siga estas diretrizes.

## 🚀 Como começar?

1. Faça um **Fork** do projeto.
2. Clone o seu fork: `git clone https://github.com/ali203939/GestaoDespesas.git`.
3. Crie uma **Branch** para sua funcionalidade: `git checkout -b feature/nova-funcionalidade`.

## 🛠️ Padrões de Desenvolvimento

Para garantir que o projeto continue robusto, siga estes passos antes de enviar um Pull Request:

### 1. Qualidade de Código (Linting)
Sempre rode o linter para garantir que o código segue os padrões do TypeScript:
```bash
npm run lint
```
